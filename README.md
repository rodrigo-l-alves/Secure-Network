# Secure-Network

Advanced Network Security and Architectures (SAAR) project focused on designing, attacking, and securing network infrastructures using real-world protocols and defense mechanisms.

---

##  Overview

This repository contains multiple hands-on labs exploring network security from both **offensive and defensive perspectives**. The project was developed using simulated environments (GNS3, Linux VMs) and focuses on:

- Network attacks and vulnerabilities
- Secure network design and configuration
- Traffic analysis and protocol validation
- High availability and redundancy mechanisms

---
## 🔐 Key Topics Covered

### Network Attacks
- MAC Table Overflow  
- DHCP Spoofing  
- ARP Poisoning (Man-in-the-Middle)  
- DNS Spoofing  
- RIP Poisoning  

### Defensive Mechanisms
- Port Security  
- DHCP Snooping  
- Dynamic ARP Inspection (DAI)  
- Firewall configuration (iptables / Cisco ACLs)  

### Secure Network Protocols
- IPSec (IKEv1, IKEv2, AH, ESP)  
- DMVPN (Phase 3)  
- BGP/MPLS L3VPNs  
- PKI and certificate-based authentication  

### High Availability & Load Balancing
- HSRP (Hot Standby Router Protocol)  
- GLBP (Gateway Load Balancing Protocol)  
- Server Load Balancing (SLB)  
- Failover and redundancy testing  

### Advanced Networking
- Docker networking (macvlan, VXLAN)  
- Network namespaces and tunneling  

---

## Technologies Used

- **GNS3** – Network simulation  
- **Cisco IOS** – Router and switch configuration  
- **Wireshark** – Packet analysis  
- **Linux (Kali/Ubuntu)** – Attack simulation and tools  
- **Docker** – Network virtualization  
- **Python / Scapy** – Traffic manipulation (attacks)  

---

## Highlights

- Implemented and analyzed real-world network attacks and their mitigations  
- Designed secure communication using IPSec with both symmetric and PKI-based authentication  
- Validated encryption and protocol behavior through packet-level inspection  
- Built resilient network architectures with redundancy and load balancing  
- Simulated enterprise-grade environments with multiple routers and hosts
